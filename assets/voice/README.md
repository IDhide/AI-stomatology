# Образец голоса для клонирования (Fish Speech / XTTS / Qwen3-TTS-VC)

Сюда кладётся образец голоса, которым будет говорить Оливия.

## Что нужно

1. **`reference.wav`** — образец голоса.
   - 10–15 секунд **чистой** речи (без музыки, шума, эха).
   - Моно, желательно 16–24 кГц WAV. MP3 тоже примет, но WAV надёжнее.
   - Конвертация из mp3:
     ```bash
     ffmpeg -i reference.mp3 -ac 1 -ar 24000 reference.wav
     ```

2. **`reference.txt`** — точная **расшифровка** того, что произнесено в образце.
   Слово в слово, обычным текстом, в одну-две строки. Для voice cloning это
   обязательно — модель сопоставляет звук и текст, чтобы извлечь тембр.

## Приватность

Файлы голоса **не коммитятся** в репозиторий (см. `.gitignore`) — это личные
данные. На рабочем киоске положите их в этот каталог вручную; он монтируется
в контейнер как `/app/assets/voice`.

## Запуск

```bash
make fish       # живой голос Fish Speech / OpenAudio (лучшее качество, GPU)
make fish-cpu   # то же без GPU (медленнее)
make xtts       # лёгкое клонирование XTTS v2
make qwen       # Qwen3-TTS-VC
```

> Fish Speech и XTTS принимают и `reference.mp3`, и `reference.wav`.
> Для Qwen нужен именно `reference.wav` + расшифровка в `reference.txt`.
