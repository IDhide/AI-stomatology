# Видео-заставка (медузы)

Положите сюда файл **`jellyfish.mp4`** — киоск автоматически покажет его
вместо процедурных (рисованных) медуз, зациклит и приглушит звук.

- Файл не коммитится в git (см. `.gitignore`) — кладётся вручную.
- Для вертикального телевизора берите вертикальное видео (9:16),
  оно растянется на весь экран (`object-fit: cover`).
- Хорошие бесплатные источники: Pexels, Pixabay (запрос «jellyfish vertical»).
- Рекомендуемый формат: H.264 MP4, 1080×1920, 24–30 fps, до ~50 МБ.

Конвертация/обрезка под вертикаль, если видео горизонтальное:

```bash
ffmpeg -i input.mp4 -vf "crop=ih*9/16:ih,scale=1080:1920" -an -t 60 jellyfish.mp4
```

Если файла нет — рисуются процедурные медузы (canvas), киоск работает как обычно.
